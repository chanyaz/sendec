Get offset of articles

https://insydia.com/api/news/
Parameter Value
token token_value
offset amount of news
Required
cat ID of the category
Not required
example:
https://insydia.com/api/news/token=eyJ0eXAiOiJKV1QiLCJhbGciOiJIUzI1NiJ9 ... pqUgeGCZvjTi7c5ANjij25s3dDhsV9xLBQv5LLCe28I&cat=1&offset=10
will return 10 news of the category with ID = 1 ('Technology')
